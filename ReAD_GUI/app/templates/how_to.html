With this web application you can run the ReAD computer vision models to:
  • display images classification
  • show image classification as JSON
  • show image classification as RDF
Display images classification
Push "Browse" and select one or more image files.
On the right select "Show images".
Press "Start Model" and wait some seconds till results are shown on your page.
The list of images with their name and classification will be displayed:
Show JSON or RDF
As show before, push "Browse" and select one or more image files.
On the right select "JSON" or "RDF".
Press "Start Model" and wait some seconds till results are shown on your page.
